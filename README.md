
# 🤖 Discord Moderation Bot (Anti-Fake News & Bullying)

บอท Discord สำหรับตรวจสอบข้อความ **คำหยาบ / การบูลลี่ / ข่าวปลอม**  
เชื่อมต่อกับ **Typhoon AI API** เพื่อวิเคราะห์ข้อความแบบ Real-time  
หากพบข้อความที่เข้าข่ายผิดกฎ จะ **แจ้งเตือนและลบข้อความทันที**

---

## 🚀 ฟีเจอร์
- ตรวจจับคำหยาบและการบูลลี่ด้วย AI
- ลบข้อความที่ผิดกฎโดยอัตโนมัติ
- แจ้งเตือนในช่องแชทเมื่อมีการลบข้อความ
- รองรับการปรับแต่ง Threshold (ระดับความเข้มงวด)
- สามารถขยายต่อไปยังระบบตรวจจับข่าวปลอมและภาพ/วิดีโอ
